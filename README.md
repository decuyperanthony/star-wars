## Getting Started

### Prerequisites

* npm
```sh
npm install npm@latest -g
```

### Installation

1. Clone the repo
```sh
git clone https://github.com/decuyperanthony/star-wars.git
```

2. To init server
```sh
cd server
cp .env.example .env
npm install
npm start
```
3. To init client
```sh
cd client
yarn
yarn start
```
4. Authentification
```
user: Luke
password: DadSucks
```
